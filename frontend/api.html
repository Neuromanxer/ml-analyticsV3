Authentication

All API requests require authentication using an API key. You can obtain your API key from your account dashboard.

API Key Security

Your API key grants access to your account and resources. Keep it secure and do not share it in publicly accessible areas like GitHub, client-side code, or in your application source code.

Authorization Header

Include your API key in the Authorization header of all requests:

Authorization: Bearer YOUR_API_KEY

Example Request

curl -X GET "https://api.mlanalytics.com/v1/models" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
